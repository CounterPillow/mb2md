mkmdtl
======

mkmdtl is a collection of utilities for generating GitHub-flavoured markdown
tables from track listing data.


Dependencies
------------

* Python 3
* requests
* wcwidth


Tools
-----

### mb2md

mb2md generates a markdown table from a MusicBrainz track listing.

#### Usage


```
mb2md MBID
```
